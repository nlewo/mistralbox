# Mistralbox

This project runs Mistral Vibe in a Bubblewrap sandbox for secure and isolated execution. It is currently dedicated for Nix projects.

To run it:

    nix run github:nlewo/mistralbox#

You can also run `bash` in the sandbox to see what is exposed to Mistral:

    nix run github:nlewo/mistralbox# -- --bash
